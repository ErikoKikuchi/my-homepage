Concept

このサイトは「再現」よりも「理解」を大切にしています。

回数をこなすことより、自分の身体を観察すること。
「効いた感覚」よりも「わかった」という変化を。
指示に従うだけでなく、自分で調整できるようになること。

身体には軸があります。
その軸を感じ、滑らかに使えるようになるまでには、
感覚と原理を丁寧に接続していく時間が必要です。

外側の形を真似るのではなく、内側から動きを組み立てる。
原理原則と個別性の掛け算で、あなた自身の動きの地図をつくっていく。
そういう場所です。
